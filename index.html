Wordo Game Rules

Welcome to Wordo! Here are the rules:

  • The game consists of guessing a word with a specified length.
  • You have a limited number of attempts to guess the word.
  • The word will be randomly selected from a predefined list.
  • Each attempt involves selecting letters to form the word.
  • You can use the keyboard or click on the on-screen keyboard to select letters.
  • If a letter is correctly guessed, it will be displayed in the corresponding position.
  • If a letter is incorrect, an error message will be displayed.
  • You can delete letters using the backspace key.
  • Once all attempts are used, the game ends, and the correct word is revealed.

Have fun playing Wordo!
